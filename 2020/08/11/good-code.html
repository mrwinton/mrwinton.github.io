Good Code

August 11, 2020

Good code:

  1. Addresses the needs of users
    • provides value
    • is shipped
  2. Addresses the needs of maintainers
    • empowers changes
    • is reliable
  3. Is good
    • doesn’t leave the world worse off

This is an incomplete definition, but still better than the first draft:

Good Code, it depends.
